<h3> git cherry-pick [insert commit reference]<h3>
<p>Tillämpa en commits ändringar på en annan gren.<p>
<h3>git clone <repository-url><h3>
<p>Klona ett specificerat repository<p>
<h3>git clone <repository-url> <directory-name><h3>
<p>Klona ett arkiv och namnge den lokala katalogen<p>
<h3>git clone <repository-url> --origin <name><h3>
<p>Klona ett arkiv och namnge fjärrkontrollen (<namn>). Om du inte vill namnge fjärrkontrollen, kommer Git att tillhandahålla standardnamnets ursprung.<p>
<h3>git clone <repository-url> --branch <branch-name><h3>
<p>Klona ett arkiv och kolla in den specifika grenen.<p>
<h3>git clone <repository-url> --depth <depth><h3>
<p>Klona ett arkiv med ett specificerat antal commits<p>
