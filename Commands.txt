<h3> git branch </h3>
<p> med git branch kan man se alla lokala branches som existerar i sin git repository</p>
<h3> git branch -a </h3>
<p> med git branch -a kan man se alla lokala och o-lokala branches i sin git repository</p>
<h3> git branch -c </h3>
<p> git branch -c kan man använda ifall man skulle vilja kopera en branch</p>
<h3> git branch -d <branch-name> </h3>
<p> git branch -d <branch-name> används för att delete:a en lokal branch, men kommer inte funka om det finns icke mergat material</p>
<h3> git branch -m <branch-name> <new-branch-name> </h3>
<p> detta git kommand används för att byta namn på sin branch, kan användas om det blir konflikter vid merge eller liknande</p>
<h3> git branch -r </h3>
<p> används för att see alla branches som inte är lokala</p>
<h3> git push <remote> --delete <remote-branch-name> </h3>
<p> detta kommando används ifall man skulle vilja ta bort en icke lokal branch </p>
<h3> git push --set-upstream <remote> <branch> </h3>
<p> detta kommando används för att skappa en upstream branch, när dett kommando körs kommer din lokala branch pushas till en ny remote branch</p>
<h3> git checkout <branch-name> </h3>
<p> detta kommando avänds för att byta från en branch till en annan </p>
