<h3> git branch </h3>
<p> med git branch kan man se alla lokala branches som existerar i sin git repository</p>
<h3> git branch -a </h3>
<p> med git branch -a kan man se alla lokala och o-lokala branches i sin git repository</p>
<h3> git branch -c </h3>
<p> git branch -c kan man använda ifall man skulle vilja kopera en branch</p>
<h3> git branch -d <branch-name> </h3>
<p> git branch -d <branch-name> används för att delete:a en lokal branch, men kommer inte funka om det finns icke mergat material</p>
